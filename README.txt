Automatically generate featured image for your blog post from your post title. All you need is to enter the API key of Artsmart and you are good to go. Other configurations are optional.

== Description ==

The KH Featured AI Image Generator provides an automated way to produce stunning featured images for your blog posts without any fuss. All you need is an Artsmart API key and you will have beautiful imagery tailored to the post's title in no time. Optional configuration settings allow you to further personalize your images too!

Here is a summary of what you can do:

*   Automate your featured image creation using AI text to image
*   Make your website more appealing to visitors to read
*   Easy customisation with simple user interface
*   No coding required

If you need any assistance, feel free to contact me at [KnowHalim](https://knowhalim.com/contact-me "Contact Halim")

== Installation ==

To install, you just need to follow these steps

1. Upload `kh-auto-ai-image-generator` zip file to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Navigate to the Settings page and configure.
